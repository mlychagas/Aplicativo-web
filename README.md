## Sobre o Projeto

Aplicação desenvolvida como parte da disciplina **Programação Web 2**. Utiliza CSS, JavaScript, EJS e TypeScript. O objetivo é ensinar conceitos básicos e intermediários de desenvolvimento web fullstack.

## Instalação

```bash
# Clone o repositório
git clone https://github.com/mlychagas/Aplicativo-web.git

# Acesse o diretório
cd Aplicativo-web

# Instale as dependências
npm install

# Inicie o aplicativo
npm start
```

## Como Usar

1. Após instalar as dependências, rode o servidor (`npm start`).
2. Abra o navegador e acesse [http://localhost:3000](http://localhost:3000).
3. Explore as funcionalidades da aplicação.
